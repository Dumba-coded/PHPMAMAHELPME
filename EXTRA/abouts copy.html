About CityRide

CityRide is a student-built project developed by Fatima and Ashley. This is our first full rental system created with PHP, MySQL, and phpMyAdmin.

Our Mission

To make vehicle rentals fast, simple, and reliable — whether you're searching for a short trip, daily commute, or weekend adventure.

Why We Built This

CityRide started as a challenge to build a real working system — not just a webpage. Through it, we learned interface design, user accounts, data handling, and how real rental platforms work behind the scenes.
