About Us

CityRide is a project proudly created by Fatima and Ashley. This is our first major web programming project using PHP, MySQL, and phpMyAdmin on a local host environment.

Our goal was to create a clean and easy-to-use rental service system while learning how real websites manage user accounts, vehicle listings, and customer interactions.

Our Mission

CityRide aims to make vehicle rentals simple, fast, and accessible for everyone. Whether you're looking for a car, bike, or scooter, our system lets you browse options and plan your trip with ease.

Why We Created This

As students learning web development, we wanted to challenge ourselves with a complete project — not just a webpage. This meant designing user interfaces, connecting forms to a database, and making sure the system works smoothly.
